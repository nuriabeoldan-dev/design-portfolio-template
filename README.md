# Bienvenido a mi portfolio de diseño gráfico!

>Aquí puedes encontrar mi sitio en vivo! [https://nuriabeoldan-dev.github.io/design-portfolio-template/](https://nuriabeoldan-dev.github.io/design-portfolio-template/)

### Descripción
Este proyecto consiste en un portfolio web personal de diseño gráfico,
desarrollado con HTML, CSS y JavaScript básico.

### Objetivo
El objetivo del proyecto es crear un portfolio profesional que muestre mis
proyectos de diseño gráfico, mis habilidades de diseño y maquetación web y mi perfil personal,
siguiendo una estructura clara y una estética cuidada.

### Tecnologías utilizadas
- HTML5
- CSS3 (estructura modular por carpetas)
- JavaScript básico

### Secciones del portfolio
- Hero con animaciones
- Sobre mí
- Software y habilidades
- Proyectos
- Contacto
