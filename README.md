# Bienvenido a mi portfolio de diseño gráfico!

>Aquí puedes encontrar mi sitio en vivo! [https://nuriabeoldan-dev.github.io/design-portfolio-template/](https://nuriabeoldan-dev.github.io/design-portfolio-template/)

### Descripción

Este repositorio contiene mi portfolio personal desarrollado como proyecto académico. El objetivo principal ha sido crear una base sólida, clara y escalable que represente mi identidad como diseñadora y que pueda evolucionar con el tiempo.
El portfolio está diseñado como una página web de desplazamiento vertical (scroll), donde se presentan diferentes secciones:
hero, sobre mí, proyectos, habilidades, software y contacto.

La estética busca un equilibrio entre sobriedad, impacto visual y claridad, inspirándose, además de en mi propia estética personal - como la que utilizo en mi logotipo y CV - en portfolios profesionales del ámbito del diseño gráfico y web, como por ejemplo la página de Adham Dannaway (https://www.adhamdannaway.com/). 

### Objetivos

- Aplicar conocimientos básicos de HTML, CSS y JavaScript.
- Crear una estructura clara y ordenada de archivos.
- Desarrollar un diseño coherente y personal.
- Practicar animaciones sencillas y efectos visuales.
- Priorizar la legibilidad, la jerarquía visual y la experiencia de usuario.

### Tecnologías utilizadas

- HTML5
- CSS3 (organizado por carpetas: base, layout, components, index, otros)
- JavaScript básico (interacciones y animaciones simples)
- Fuentes web (Google Fonts)
- Diseño responsive mediante media queries


## Estado del proyecto

Este proyecto **no se considera una versión final cerrada**, sino una plantilla viva.
Está pensada como un punto de partida que irá creciendo y mejorando con nuevas
iteraciones, ajustes visuales y mejoras técnicas.


## Metodología de trabajo (toque atelier)

Este portfolio se ha desarrollado siguiendo una filosofía de **aprender haciendo**.
Muchas decisiones se han tomado probando, equivocándome y ajustando sobre la marcha.

El diseño web es un proceso iterativo: cada versión mejora la anterior, y cada error aporta aprendizaje.

Esta plantilla es como una red que se irá remendando, afinando y reforzando con el tiempo y la experiencia.
